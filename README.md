# renq
Rename a sequence of file starting from a number and bring it to 1.

## LIMITATION
* you must copy paste the output into a script and run it.

##     ===SYNOPSIS===
```shell
# renq [startNo] [endNo] [ext] ([pad] [mv cmd])
# ex.
renq 60 89 jpg
renq 60 89 jpg 4
renq 60 89 jpg 4 "git mv"
renq 60 89 jpg 4 "mv"
```
